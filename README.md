## üìå Project Overview

**VADSAC Academy Quiz App** is a mobile quiz application developed as the final project for **Mobile Programming 1**. The app showcases modern Android development practices and features including:

- Kotlin-based Android development  
- MVVM architecture  
- Kotlin Coroutines for background processing  
- Navigation components  
- Local data handling for quizzes  
- Jetpack Compose UI  
- State management and error handling  

The app provides multiple-choice quizzes where users can test their knowledge, answer timed questions, and receive scores at the end of each session.
